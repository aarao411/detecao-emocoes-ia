# detecao-emocoes-ia
Sistema de detecção de emoções faciais em tempo real utilizando visão computacional (MediaPipe), deep learning (TensorFlow/Keras) e o modelo FER-2013. O projeto identifica 7 emoções (feliz, triste, raiva, surpresa, nojo, medo e neutro) com alta precisão e exibe o resultado ao vivo pela webcam.
